# 🚀 วิธีใช้งานและย้ายเครื่อง (Offline POS 100%)

ระบบนี้ถูกออกแบบมาเป็น **"แอปพลิเคชันพกพา (Portable App)"** ที่บันทึกข้อมูลลงไดรฟ์โดยตรง คุณสามารถย้ายไปรันบนคอมพิวเตอร์เครื่องไหนก็ได้ที่มี Node.js ครับ

## 📦 1. วิธีเตรียมไฟล์เพื่อย้ายเครื่อง
1.  ทำการ **Zip (บีบอัด)** โฟลเดอร์ `test` ทั้งหมด (รวมถึงไฟล์ `local_database.json`)
2.  ส่งไฟล์ Zip นี้ไปยังคอมพิวเตอร์เครื่องใหม่ (ผ่าน USB, LINE, หรือ Google Drive)
3.  ทำการ **Extract (แตกไฟล์)** ลงในคอมพิวเตอร์เครื่องใหม่ในตำแหน่งที่ต้องการ

## ⚙️ 2. สิ่งที่เครื่องใหม่ต้องมี
*   **Node.js**: ต้องติดตั้ง Node.js (แนะนำ v18 ขึ้นไป) สามารถโหลดได้ที่ [nodejs.org](https://nodejs.org/)

## 🚀 3. วิธีเปิดใช้งาน (ในเครื่องใหม่)
1.  เปิดโฟลเดอร์โครงการที่ย้ายมา
2.  (ครั้งแรกเท่านั้น) ดับเบิลคลิกไฟล์ `install_deps.bat` (เดี๋ยวผมจะสร้างให้ครับ) เพื่อติดตั้ง Library ที่จำเป็น
3.  **ดับเบิลคลิกไฟล์ `run_minimart.bat`**
    *   ไฟล์นี้จะเปิดทั้ง **หน้าจอฐานข้อมูล (Disk Server)** และ **ตัวแอป POS** ให้พร้อมกันโดยอัตโนมัติครับ
4.  หน้าจอ Browser จะเปิดขึ้นมาที่ `http://localhost:5173` (หรือตามพอร์ตที่ระบุ) กราฟและข้อมูลจะเหมือนเดิม 100% ครับ

## 💾 4. ข้อมูลอยู่ที่ไหน?
*   ทุกอย่างที่คุณขายจะถูกเก็บไว้ที่ไฟล์ `local_database.json`
*   **แนะนำ:** ควรหมั่นก๊อปปี้ไฟล์นี้เก็บไว้ที่อื่นเพื่อเป็นสำรอง (Backup) ทุกๆ สิ้นวันครับ
